Wikimedia
===========
The wikimedia package contains functionality related
to the use cases for the Bitu IDM at the Wikimedia Foundation.

Any functionality that can not be generalised sufficiently to
be of use in other organisations should be fitted into this
package.

Other organisations can use the package as inspiration and are
encouraged to create a similar package for their own unique
use cases.
